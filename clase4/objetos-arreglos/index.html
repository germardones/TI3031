Métodos Funcionales Avanzados

Más allá de map y filter, JavaScript tiene una batería de métodos para transformar y analizar colecciones de forma declarativa.

reduce() avanzado

No solo suma — puede transformar un array en cualquier estructura: objeto, Map, string, otro array.

// Agrupar por propiedad
const byDept = empleados.reduce((acc, emp) => {
  acc[emp.dept] = acc[emp.dept] || [];
  acc[emp.dept].push(emp);
  return acc;
}, {});

find / findIndex

find() retorna el primer elemento que cumple la condición (o undefined). findIndex() retorna su posición (o -1). Detienen la búsqueda al encontrar el primero.

const admin = users.find(u => u.role === "admin");
const idx = users.findIndex(u => u.id === 42);
// Para actualizar en el array:
if (idx !== -1) users[idx] = { ...users[idx], active: true };

some / every

Retornan booleano. some() = ¿al menos uno cumple? every() = ¿todos cumplen? Útiles para validaciones.

const hayAdmin = users.some(u => u.role === "admin");
const todosActivos = users.every(u => u.active);
const formOK = inputs.every(i => i.value.trim() !== "");

flat / flatMap

flat() aplana arrays anidados. flatMap() mapea y aplana en un solo paso — más eficiente que .map().flat().

[[1,2],[3,4]].flat(); // [1,2,3,4]

// Expandir cada usuario en sus roles
users.flatMap(u => u.roles);

Map y Set: Colecciones Especializadas

Map: Diccionario tipado

A diferencia de un objeto, las claves pueden ser cualquier tipo — objetos, funciones, primitivos. Ideal para cachés y asociaciones complejas.

const cache = new Map();
cache.set(userObj, "perfil-data");
cache.set("config", { theme: "dark" });

cache.get(userObj); // "perfil-data"
cache.has("config"); // true
cache.size; // 2

for (const [key, val] of cache) {
  console.log(key, val);
}

Set: Colección de únicos

Almacena valores sin duplicados. El caso de uso clásico: eliminar duplicados de un array en una línea.

const tags = new Set(["js", "css", "js"]);
tags.size; // 2 — ignora duplicado
tags.add("html");
tags.has("css"); // true

// Eliminar duplicados de array:
const unique = [...new Set(arr)];

// Intersección de dos arrays:
const common = a.filter(x => setB.has(x));

Clonación: Superficial vs Profunda

El operador spread {...obj} hace una copia superficial. Para estructuras anidadas necesitas una copia profunda.

⚠️ El Bug del Spread
const orig = { a: 1, nested: { b: 2 } };
const copy = { ...orig };

copy.a = 99; // OK — primitivo copiado
copy.nested.b = 99; // BUG!
console.log(orig.nested.b); // 99 — también cambió!
✅ structuredClone()

API nativa del navegador (2022). Copia profunda sin dependencias externas. Soporta Date, Map, Set, RegExp.

const copy = structuredClone(orig);
copy.nested.b = 99;
console.log(orig.nested.b); // 2 — intacto

JSON.parse(JSON.stringify(obj)); // pierde Date, undefined, funciones
