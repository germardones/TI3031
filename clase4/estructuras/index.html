Evaluación Logicom

Operador OR ( || )

Excelente para valores por defecto. Si el primer valor es falsy (null, undefined, false, 0, ""), JS toma el segundo.

Operador AND ( && )

Útil para ejecuciones condicionales. El código a la derecha solo corre si el de la izquierda es truthy.
