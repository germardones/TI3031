Contextos de Ejecución

Resolución de Scopes y la arquitectura del motor de compilación Just-In-Time.

1. El Concepto de Contexto

Antes de que JavaScript ejecute una sola línea de código, el motor crea un Execution Context. Este es el entorno que contiene toda la información necesaria para ejecutar el código actual: variables, funciones y el valor de this.

Tipos de Contextos

Global Context

El entorno base que se crea al iniciar el script. Es el "padre" de todos los demás.

Functional Context

Cada vez que invocas una función, se genera un nuevo contexto privado e independiente.

2. Las Fases de Vida

Fase de Creación

El motor escanea el código, reserva espacio en memoria para variables y funciones. Aquí es donde ocurre el Hoisting: las declaraciones de var se inicializan como undefined.

Fase de Ejecución

El motor procesa el código línea por línea. Asigna valores reales a las variables calculando expresiones y ejecutando las llamadas a funciones.

3. The Scope Chain

Cuando una función necesita una variable que no está en su entorno actual, JavaScript utiliza la Scope Chain. Esta cadena conecta el entorno actual con todos sus entornos padres hasta llegar al Global.
