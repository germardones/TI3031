# Programación Front End (TI3031)

## Descripción del Proyecto

Este repositorio actúa como el nodo central o *hub* de aprendizaje para el curso **Programación Front End (TI3031)**. Nuestro objetivo es proporcionar un entorno estructurado, moderno y altamente interactivo para que los estudiantes adquieran y consoliden los conocimientos fundamentales del desarrollo web.

Mediante este material, los participantes podrán explorar desde la estructuración semántica en HTML5, la estilización avanzada con CSS3, hasta la implementación de lógica en el navegador mediante JavaScript, preparándose así para los estándares actuales de la industria tecnológica.

---

## Estructura de Clases

El material está dividido de manera progresiva. A medida que avanza el semestre, las clases correspondientes se irán desbloqueando en la plataforma principal.

1. **Clase 01: Conociendo HTML (Disponible)**
   - Herramienta interactiva para analizar en tiempo real las etiquetas y componentes semánticos fundamentales (header, nav, main, section, footer, etc.) y su correcta disposición dentro de un documento.

2. **Clase 02: CSS Avanzado y Frameworks (Próximamente)**
   - Uso de las nuevas Propiedades de CSS3 y la integración de Frameworks CSS para potenciar el diseño visual.

3. **Clase 03: Control de Versiones e IA (Próximamente)**
   - Aprender a trabajar colaborativamente con Git y explorar el uso de Inteligencia Artificial (GitHub Copilot) en el desarrollo de software.

## Instrucciones de Uso

1. Abra el archivo `index.html` en cualquier navegador web moderno (Google Chrome, Mozilla Firefox, Microsoft Edge o Safari).
2. Explore los módulos habilitados haciendo clic en sus respectivas tarjetas.
3. Para la **Clase 01**, navegue por el documento y pase el cursor (`hover`) sobre los respectivos contenedores para revelar en tiempo real la etiqueta `HTML` en uso.
