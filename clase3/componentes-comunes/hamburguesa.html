Menú Hamburguesa

El ícono de tres líneas que se transforma en una "X" al abrirse. Aprende la animación CSS y el control JavaScript que lo hacen funcionar.

Las 3 barras se transforman en una ✕ y el menú aparece desde la derecha. Todo hecho con CSS puro + 5 líneas de JS.

Código HTML — Línea a Línea

Estructura del Botón y Menú

hamburguesa.html 

<button
  class="hamburger"
  id="menuBtn"
  aria-label="Abrir menú"
  aria-expanded="false">

  
  <span class="barra"></span>
  <span class="barra"></span>
  <span class="barra"></span>
</button>


<nav
  class="menu-movil"
  id="menu"
  aria-hidden="true">
  <ul>
    <li><a href="#">Inicio</a></li>
    <li><a href="#">Sobre mí</a></li>
    <li><a href="#">Proyectos</a></li>
    <li><a href="#">Contacto</a></li>
  </ul>
</nav>
1

<button> Usamos button y no div o span. Los elementos <button> son enfocables con Tab, activables con Enter/Space y anunciados por lectores de pantalla. Usar un <div> nos obliga a replicar todo eso manualmente.

1b

aria-label="Abrir menú" Descripción para lectores de pantalla. El botón visualmente solo muestra 3 rayas — sin este atributo, un usuario ciego escucharía solo "botón" sin saber qué hace. aria-label le da contexto.

1c

aria-expanded="false" Estado de accesibilidad. Indica si el menú está abierto o cerrado. JavaScript debe actualizar este valor a "true" cuando el menú se abre. Los lectores de pantalla lo anuncian al usuario.

2

<span class="barra"></span> ×3 Las tres barras del ícono. Son <span> vacíos que se convierten en líneas con CSS (width, height, background). CSS los anima individualmente con :nth-child().

3

aria-hidden="true" en el menú Oculta el menú para lectores de pantalla cuando está cerrado. Cuando JavaScript lo abra, debe cambiar este valor a "false" para que el contenido sea accesible.

Código CSS — La Animación Explicada

Transformar 3 barras en una ✕

hamburguesa.css 
/* ── BOTÓN CONTENEDOR ────────────────── */
.hamburger {
  display: flex;            /* [A] Barras en columna */
  flex-direction: column;  /* [B] Una debajo de la otra */
  gap: 6px;                /* [C] Espacio entre barras */
  background: none;        /* [D] Sin fondo */
  border: none;            /* [E] Sin borde */
  cursor: pointer;        /* [F] Cursor de mano */
  padding: 8px;
}

/* ── CADA BARRA ──────────────────────── */
.barra {
  width: 28px;             /* [G] Ancho fijo */
  height: 3px;             /* [H] Solo 3px de alto */
  background: #f8fafc;    /* [I] Color blanco */
  border-radius: 4px;     /* [J] Bordes redondeados */
  transition: all 0.35s cubic-bezier(0.4, 0, 0.2, 1); /* [K] */
  display: block;          /* [L] Necesario para transform */
}

/* ── ESTADO ABIERTO: barra 1 → / ──── */
.hamburger.open .barra:nth-child(1) {
  transform: translateY(9px) rotate(45deg); /* [M] */
}

/* ── ESTADO ABIERTO: barra 2 → invisible */
.hamburger.open .barra:nth-child(2) {
  opacity: 0;              /* [N] Desaparece */
  transform: scaleX(0);   /* [O] Se encoge */
}

/* ── ESTADO ABIERTO: barra 3 → \ ──── */
.hamburger.open .barra:nth-child(3) {
  transform: translateY(-9px) rotate(-45deg); /* [P] */
}

/* ── MENÚ LATERAL ────────────────────── */
.menu-movil {
  position: fixed;         /* [Q] Fuera del flujo normal */
  top: 0;
  right: -300px;           /* [R] Oculto a la derecha */
  width: 300px;
  height: 100vh;           /* [S] Altura completa */
  background: #1e293b;
  transition: right 0.35s ease; /* [T] Animación de deslizamiento */
  padding: 5rem 2rem;
}
.menu-movil.open {
  right: 0;               /* [U] Entra en pantalla */
}
A-B

display:flex + flex-direction:column Apila las 3 barras verticalmente. Por defecto flex pone los elementos en fila. Con column los pone en columna: barra arriba, barra en medio, barra abajo.

C

gap: 6px Separa las barras exactamente 6 píxeles. Este valor es importante para la animación: al transformar las barras 1 y 3, las moveremos 9px (= la barra de 3px + el gap de 6px) para que se crucen en el centro.

K

transition: all 0.35s cubic-bezier(0.4, 0, 0.2, 1) La clave de toda la animación. Cada cambio de propiedad (transform, opacity) se animará en 350ms con una curva de aceleración suave. Sin esto, la transformación sería instantánea.

M

translateY(9px) rotate(45deg) La barra superior se mueve 9px hacia abajo (para alinearse con el centro) y luego rota 45°. Esto la convierte en la parte "/" de la X. El orden importa: primero se mueve, luego rota.

N-O

opacity:0 + scaleX(0) La barra del medio desaparece de dos maneras: se vuelve transparente (opacity) y se encoge horizontalmente (scaleX). Usar los dos juntos hace la animación más fluida.

P

translateY(-9px) rotate(-45deg) La barra inferior se mueve 9px hacia arriba y rota -45°. Esto la convierte en la parte "\" de la X. Combinada con la barra superior, forman la ✕ perfecta.

R

right: -300px Oculta el menú "fuera de pantalla" a la derecha. El menú tiene 300px de ancho y está posicionado -300px, así que queda completamente invisible. Cuando añadimos la clase .open, right cambia a 0 y el menú "entra".

T

transition: right 0.35s ease Anima el movimiento del menú. Cuando JavaScript cambia right de -300px a 0, CSS anima esa transición suavemente en lugar de ser instantánea. El menú "desliza" desde afuera.

JavaScript — El Control del Toggle

Activar y desactivar el menú

hamburguesa.js 
// 1. Seleccionamos los elementos del DOM
const btn  = document.getElementById('menuBtn');
const menu = document.getElementById('menu');

// 2. Escuchamos el clic en el botón
btn.addEventListener('click', () => {

  // 3. Alternamos la clase 'open' en el botón y el menú
  btn.classList.toggle('open');
  menu.classList.toggle('open');

  // 4. Actualizamos los atributos de accesibilidad
  const isOpen = menu.classList.contains('open');
  btn.setAttribute('aria-expanded', isOpen);
  menu.setAttribute('aria-hidden', !isOpen);
});
1

getElementById() Obtiene referencias a los elementos HTML. En lugar de buscar los elementos cada vez que el usuario hace clic, los guardamos en variables (btn, menu) para acceder a ellos rápidamente.

2

addEventListener('click', ...) Registra una función que se ejecuta cuando el usuario hace clic. Esto es un patrón fundamental de JavaScript: "cuando pase X evento, ejecuta Y función".

3

classList.toggle('open') El comando más importante de esta lógica. Si el elemento tiene la clase open, la quita. Si no la tiene, la agrega. Una sola línea reemplaza un condicional if/else completo.

4

setAttribute('aria-expanded', isOpen) Mantiene los atributos de accesibilidad actualizados. classList.contains() devuelve true/false, que usamos para actualizar aria-expanded y aria-hidden automáticamente.
