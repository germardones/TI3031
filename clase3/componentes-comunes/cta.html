Código HTML — Línea a Línea

Estructura de la Sección CTA

cta.html 

<section class="cta-seccion">

  
  <span class="cta-badge">Disponible ahora</span>

  
  <h2 class="cta-titulo">
    ¿Listo para llevar tu proyecto al siguiente nivel?
  </h2>

  
  <p class="cta-desc">
    Transformamos tus ideas en experiencias digitales...
  </p>

  
  <div class="cta-botones">
    <a href="contacto.html" class="btn-primario">Comenzar ahora →</a>
    <a href="portfolio.html" class="btn-secundario">Ver portfolio</a>
  </div>

</section>
1

<section> Etiqueta semántica para un bloque temático independiente. Una CTA es una sección completa con sentido propio. Los motores de búsqueda y lectores de pantalla la reconocen como una unidad de contenido.

2

<span class="cta-badge"> Etiqueta contextual pequeña sobre el título. El badge aporta urgencia o exclusividad ("Disponible ahora", "Oferta limitada"). Usar <span> es correcto porque no es un encabezado ni párrafo, solo texto decorativo en línea.

3

<h2> para el título No uses <h1> dentro de una sección. El <h1> es único por página. Las secciones usan <h2>. Esto mantiene la jerarquía correcta para SEO y accesibilidad.

5

<div class="cta-botones"> Contenedor para agrupar los botones. Al agruparlos podemos centrarlos juntos con Flexbox. Siempre incluye un botón primario (acción deseada) y opcionalmente un secundario (alternativa menos comprometida).

Código CSS — Propiedad a Propiedad

Estilos de la Sección y Botones

cta.css 
/* ── SECCIÓN CTA ─────────────────────────────── */
.cta-seccion {
  background: linear-gradient(135deg, #0f172a, #1a2744, #0f2a1f); /* [A] */
  border-radius: 1.5rem;      /* [B] Esquinas muy redondeadas */
  padding: 4rem 2.5rem;       /* [C] Espacio generoso */
  text-align: center;          /* [D] Todo centrado */
  position: relative;          /* [E] Para el pseudo-elemento ::before */
  overflow: hidden;            /* [F] Recorta el glow circular */
}

/* Efecto de luz radial en el centro */
.cta-seccion::before {
  content: '';
  position: absolute;
  top: -50%; left: 50%; transform: translateX(-50%);
  width: 600px; height: 600px;
  background: radial-gradient(circle, rgba(16,185,129,0.08), transparent 70%); /* [G] */
  pointer-events: none;        /* [H] No bloquea clics */
}

/* ── TÍTULO ──────────────────────────────────── */
.cta-titulo {
  font-size: clamp(1.75rem, 4vw, 2.5rem); /* [I] Tamaño fluido */
  font-weight: 800;
  line-height: 1.2;            /* [J] Interlineado ajustado para títulos grandes */
  margin-bottom: 1rem;
}

/* ── BOTÓN PRIMARIO ──────────────────────────── */
.btn-primario {
  background: #10b981;
  color: #ffffff;
  padding: 0.8rem 2rem;
  border-radius: 100px;
  font-weight: 700;
  text-decoration: none;
  transition: all 0.3s ease;
  display: inline-block;      /* [K] Para que padding funcione en <a> */
}
.btn-primario:hover {
  transform: translateY(-3px); /* [L] Efecto flotante */
  box-shadow: 0 12px 28px rgba(16,185,129,0.45); /* [M] Glow */
}

/* ── BOTÓN GRADIENTE ─────────────────────────── */
.btn-gradiente {
  background: linear-gradient(135deg, #10b981, #3b82f6); /* [N] */
  position: relative;
  overflow: hidden;            /* [O] Recorta el ::before */
}
.btn-gradiente::before {         /* Gradiente oscuro oculto */
  content: '';
  position: absolute; inset: 0;
  background: linear-gradient(135deg, #059669, #2563eb);
  opacity: 0;
  transition: opacity 0.3s;    /* [P] Solo anima opacidad */
}
.btn-gradiente:hover::before { opacity: 1; }  /* [Q] Revela el gradiente oscuro */
A

linear-gradient(135deg, ...) Gradiente diagonal (135° = esquina superior-izquierda → inferior-derecha). Puede recibir múltiples colores. Los degradados crean profundidad sin imágenes y son más ligeros para el navegador.

E-F

position: relative + overflow: hidden El dúo para controlar pseudo-elementos. El padre necesita relative para que el ::before absolute se posicione dentro de él. overflow: hidden recorta todo lo que salga del borde.

G

radial-gradient(circle, rgba(16,185,129,0.08), transparent 70%) Círculo de luz sutil. El gradiente radial va del color del acento (8% opacidad) hacia transparente. Crea la ilusión de una fuente de luz detrás del título, muy común en diseño dark mode.

I

clamp(1.75rem, 4vw, 2.5rem) Tamaño de fuente fluido y responsivo. La función clamp(min, ideal, max) hace que el tamaño crezca con la pantalla (4vw) pero nunca baje de 1.75rem ni suba de 2.5rem. Sin media queries.

J

line-height: 1.2 Interlineado comprimido para títulos grandes. En textos de cuerpo usamos ~1.6. En títulos grandes, la separación estándar se ve excesiva — 1.2 da un look más elegante y compacto.

K

display: inline-block Necesario para que el padding funcione correctamente en elementos <a>. Los <a> son inline por defecto; en modo inline, padding vertical no afecta el flujo. inline-block lo arregla.

N-Q

Técnica del pseudo-elemento para animar gradientes. CSS no puede animar background-gradient directamente. La solución es poner el gradiente oscuro en ::before con opacity:0 y animarlo a opacity:1 al hacer hover. Esto es más suave que cambiar el gradiente.

Jerarquía de Botones — Regla de Oro

En una misma pantalla, la jerarquía visual debe guiar al usuario hacia la acción correcta:

PRIMARIO Contratar ahora → Solo 1 por sección. La acción que quieres que el usuario tome.
SECUNDARIO Ver más → Alternativa menos comprometida.
GHOST Cancelar → Acción destructiva o de bajo valor.
