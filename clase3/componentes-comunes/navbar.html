Código HTML — Línea a Línea

Estructura del Navbar

navbar.html 

<nav class="navbar">

  
  <a href="index.html" class="logo">MiSitio</a>

  
  <ul class="navbar-links">
    <li><a href="#">Inicio</a></li>
    <li><a href="#">Sobre</a></li>
    <li><a href="#">Proyectos</a></li>
    <li><a href="#">Blog</a></li>
  </ul>

  
  <a href="contacto.html" class="navbar-cta">Contratar</a>

</nav>
1

<nav> Etiqueta semántica de HTML5. Le dice al navegador y a los lectores de pantalla que este bloque es la zona principal de navegación del sitio. Usar <div> también funcionaría visualmente, pero perdemos semántica y accesibilidad.

2

<a href="index.html"> El logo siempre es un enlace que apunta a la página principal. Usar <a> en vez de <span> es la práctica correcta — los usuarios esperan hacer clic en el logo para volver al inicio.

3

<ul> + <li> La estructura correcta para listas de navegación. Los links de un navbar son semánticamente una lista de opciones. Esto ayuda a los lectores de pantalla (accesibilidad) y a los motores de búsqueda (SEO) a entender la jerarquía.

4

.navbar-cta El botón de acción principal usa <a> y no <button> porque dirige al usuario a otra página. Se distingue visualmente del resto de los links para atraer la atención.

Código CSS — Propiedad a Propiedad

Estilos del Navbar

navbar.css 
/* ── CONTENEDOR PRINCIPAL ─────────────────── */
.navbar {
  display: flex;                    /* [A] */
  justify-content: space-between;  /* [B] */
  align-items: center;             /* [C] */
  padding: 1rem 2rem;              /* [D] */
  background: #0f172a;             /* [E] */
  position: sticky;               /* [F] */
  top: 0;                          /* [G] */
  z-index: 100;                    /* [H] */
  border-bottom: 1px solid rgba(255,255,255,0.1); /* [I] */
}

/* ── LOGO ─────────────────────────────────── */
.logo {
  font-size: 1.5rem;              /* [J] */
  font-weight: 700;               /* [K] */
  color: #10b981;                 /* [L] */
  text-decoration: none;          /* [M] */
}

/* ── LISTA DE LINKS ───────────────────────── */
.navbar-links {
  display: flex;                   /* [N] */
  list-style: none;               /* [O] */
  gap: 2rem;                      /* [P] */
  margin: 0;
  padding: 0;
}

/* ── LINKS INDIVIDUALES ───────────────────── */
.navbar-links a {
  color: #94a3b8;                 /* [Q] */
  text-decoration: none;          /* [R] */
  font-weight: 500;               /* [S] */
  transition: color 0.3s ease;   /* [T] */
}
.navbar-links a:hover { color: #f8fafc; } /* [U] */

/* ── BOTÓN CTA ────────────────────────────── */
.navbar-cta {
  background: #10b981;            /* [V] */
  color: #ffffff;
  padding: 0.5rem 1.25rem;       /* [W] */
  border-radius: 100px;          /* [X] */
  text-decoration: none;
  font-weight: 600;
  transition: all 0.3s ease;     /* [Y] */
}
.navbar-cta:hover {
  background: #059669;           /* [Z] */
  transform: translateY(-2px);   /* [AA] */
  box-shadow: 0 8px 20px rgba(16,185,129,0.4); /* [AB] */
}
A

display: flex Activa Flexbox en el navbar. Convierte al <nav> en un contenedor flex y a sus hijos directos (logo, links, CTA) en flex-items que se organizan automáticamente en una fila.

B

justify-content: space-between Distribuye los 3 elementos (logo · links · CTA) empujando el logo a la izquierda, el CTA a la derecha y los links al centro. Es el patrón más común en navbars.

C

align-items: center Alinea verticalmente todos los elementos al centro del navbar. Sin esto, los elementos quedarían en la parte superior de la barra.

D

padding: 1rem 2rem Espacio interno. El primer valor (1rem = 16px) es el espacio arriba y abajo. El segundo (2rem = 32px) es el espacio a los lados. Da "aire" al contenido del navbar.

F

position: sticky El navbar se pega al borde superior cuando el usuario hace scroll. A diferencia de position: fixed, sticky respeta el flujo del documento y no saca al elemento del layout.

G

top: 0 Define el punto de "pegado": cuando el navbar llega a 0px del borde superior de la ventana, se queda fijo ahí. Sin este valor, position: sticky no tiene efecto.

H

z-index: 100 Prioridad de apilamiento. Asegura que el navbar aparezca visualmente POR ENCIMA de todo el contenido que pase por debajo al hacer scroll. Los valores más altos = más arriba.

I

border-bottom: 1px solid rgba(255,255,255,0.1) Línea separadora sutil. El color blanco con solo 10% de opacidad crea una división casi invisible pero que da profundidad al separar el navbar del contenido.

N

display: flex en .navbar-links Pone los ítems de la lista en fila. Por defecto un <ul> es un bloque con items en columna. Flex los convierte a horizontal.

O

list-style: none Elimina los puntos (bullets) que el navegador agrega por defecto a todas las listas <ul>. En un navbar no queremos esos bullets.

T

transition: color 0.3s ease Anima el cambio de color cuando el usuario pasa el mouse. Sin esto, el cambio sería brusco e instantáneo. 0.3s = 300 milisegundos de duración. ease = arranca rápido y desacelera.

X

border-radius: 100px Hace el botón completamente redondeado (pill shape). El valor 100px es un truco: cualquier valor mayor que la mitad del alto crea bordes perfectamente circulares. Es el estilo más moderno para botones CTA.

AA

transform: translateY(-2px) Sube el botón 2 píxeles al hacer hover, creando una sensación de que el botón "flota". Este micro-movimiento da feedback visual al usuario de que el elemento es interactivo.

AB

box-shadow: 0 8px 20px rgba(16,185,129,0.4) Sombra de color (glow effect) al hacer hover. Los 4 valores son: desplazamiento-x | desplazamiento-y | difuminado | color. El color coincide con el del botón para crear un efecto de brillo.

Responsivo: ¿Y en mobile?

En pantallas pequeñas, los links del navbar no caben. La solución profesional es ocultarlos con CSS y reemplazarlos con un menú hamburguesa (ver el siguiente paso).

/* En pantallas menores a 768px */
@media (max-width: 768px) {
  .navbar-links {
    display: none; /* Ocultamos la lista de links */
  }
  .hamburger-btn {
    display: flex; /* Mostramos el botón hamburguesa */
  }
}
