Variante 1 — Feature Card

Tarjeta de Característica

card-feature.html 

<div class="card-feature">

  
  <div class="card-icon-box">⚡</div>

  
  <h3>Rendimiento</h3>
  <p>Optimizado para cargar en menos de 2 segundos.</p>

  
  <a href="#" class="card-link">Saber más →</a>

</div>
card-feature.css 
.card-feature {
  display: flex;
  flex-direction: column;          /* [A] Apila ícono, título, texto, link */
  gap: 1rem;                       /* [B] Espacio automático entre hijos */
  padding: 2rem;
  border-radius: 1.25rem;          /* [C] Esquinas redondeadas */
  border: 1px solid rgba(255,255,255,0.08); /* [D] Borde sutil */
  transition: all 0.3s ease;
}
.card-feature:hover {
  transform: translateY(-8px);     /* [E] Efecto "levitar" */
  border-color: #10b981;
  box-shadow: 0 20px 40px -15px rgba(16,185,129,0.3); /* [F] */
}

.card-icon-box {
  width: 48px; height: 48px;
  border-radius: 0.75rem;          /* [G] Cuadrado redondeado */
  background: rgba(16,185,129,0.15); /* [H] Color acento al 15% */
  display: flex;
  align-items: center;
  justify-content: center;
}

.card-feature p {
  flex-grow: 1;                   /* [I] Empuja el link al fondo */
  color: #94a3b8;
}

.card-link {
  color: #10b981;
  font-weight: 600;
  display: inline-flex;
  gap: 0.3rem;
  transition: gap 0.3s;           /* [J] Anima la flecha al hover */
}
.card-feature:hover .card-link { gap: 0.6rem; }
A

flex-direction: column Cambia el eje principal a vertical. Con column los elementos se apilan hacia abajo — ícono, título, texto, link — que es lo que queremos en una tarjeta.

B

gap: 1rem Espacio automático entre hijos. En vez de agregar margin-bottom a cada elemento, gap aplica el espacio entre todos los hijos sin afectar el primero ni el último.

E

transform: translateY(-8px) Efecto "levitar" en hover. Mueve la tarjeta 8px hacia arriba. transform no afecta el layout — los elementos vecinos no se mueven.

H

background: rgba(16,185,129,0.15) Fondo de color con baja opacidad. El mismo acento al 15% crea una "mancha de color" suave que destaca el ícono sin competir con el texto.

I

flex-grow: 1 en el párrafo. Consume todo el espacio libre disponible, empujando el link siempre al fondo de la tarjeta, independientemente de cuánto texto haya.

J

transition: gap 0.3s Anima el espacio entre el texto y la flecha. Al hacer hover la tarjeta, el gap del link pasa de 0.3rem a 0.6rem — la flecha "se aleja" sutilmente del texto.

Variante 2 — Profile Card

Tarjeta de Perfil

card-profile.html 
<div class="card-profile">

  
  <div class="profile-cover"></div>

  <div class="profile-body">

    
    <div class="profile-avatar">👩‍💻</div>

    <h3>Ana García</h3>
    <p class="profile-role">Frontend Developer</p>
    <p>Especialista en React y diseño de sistemas.</p>

    
    <div class="profile-stats">
      <div class="stat">
        <span class="stat-num">48</span>
        <span class="stat-label">Proyectos</span>
      </div>
    </div>

  </div>
</div>
card-profile.css 
.card-profile {
  border-radius: 1.25rem;
  overflow: hidden;              /* [A] El cover respeta las esquinas */
}

.profile-cover {
  height: 100px;
  background: linear-gradient(135deg, #10b981, #3b82f6); /* [B] Gradiente diagonal */
}

.profile-avatar {
  width: 64px; height: 64px;
  border-radius: 50%;           /* [C] Círculo perfecto */
  border: 3px solid #1e293b;   /* [D] Borde del color del fondo */
  margin-top: -32px;            /* [E] Truco de superposición */
}

.profile-stats {
  display: flex;
  gap: 1.5rem;
  border-top: 1px solid rgba(255,255,255,0.06); /* [F] Separador sutil */
  padding-top: 1rem;
}
.stat {
  display: flex;
  flex-direction: column;       /* [G] Número arriba, label abajo */
  align-items: center;
}
A

overflow: hidden en la tarjeta. Recorta todo lo que salga de los bordes redondeados. Sin esto, el banner de color ignoraría el border-radius de la tarjeta y "saldría" de las esquinas.

B

linear-gradient(135deg, #10b981, #3b82f6) Degradado diagonal. 135° = diagonal de arriba-izquierda a abajo-derecha. Le da personalidad al banner sin necesitar una imagen.

C

border-radius: 50% Convierte el avatar en círculo perfecto. Solo funciona en elementos cuadrados (width = height).

E

margin-top: -32px El truco del avatar superpuesto. El avatar mide 64px. Con margin-top: -32px (la mitad) sube y se superpone al banner. Los márgenes negativos sacan al elemento de su posición natural.

Variante 3 — Pricing Card

Tarjeta de Precio

card-pricing.html 
<div class="card-pricing featured">

  
  <span class="pricing-badge">Popular</span>

  <p class="plan-name">Pro</p>

  
  <div class="price">$19<span>/mes</span></div>

  
  <ul>
    <li>5 proyectos activos</li>
    <li class="disabled">Análisis avanzado</li>
  </ul>

  
  <button class="btn-plan primary">Empezar ahora</button>

</div>
card-pricing.css 
.card-pricing {
  display: flex;
  flex-direction: column;
  position: relative;           /* [A] Contexto para el badge absoluto */
  overflow: hidden;
}
.card-pricing.featured {
  box-shadow: 0 0 0 1px #10b981, /* [B] Doble sombra: borde + profundidad */
              0 20px 40px -15px rgba(16,185,129,0.3);
}

.pricing-badge {
  position: absolute;           /* [C] Sale del flujo, flota encima */
  top: 1rem; right: 1rem;
  border-radius: 100px;
}

.price { font-size: 2.75rem; font-weight: 700; } /* [D] Grande para impacto */
.price span { font-size: 1rem; color: #64748b; }  /* [E] Unidad pequeña y tenue */

.card-pricing ul {
  flex-grow: 1;                 /* [F] Empuja el botón siempre al fondo */
  list-style: none;
}
.card-pricing ul li::before { content: "✓"; color: #10b981; }   /* [G] */
.card-pricing ul li.disabled::before { content: "×"; color: #475569; } /* [H] */

.btn-plan.primary {
  background: #10b981; color: #fff;
  display: block;             /* [I] Ocupa todo el ancho */
  text-align: center;
  padding: 0.75rem;
  border-radius: 0.75rem;
}
A

position: relative en la tarjeta. Crea contexto de posicionamiento. El badge necesita position: absolute para ubicarse en la esquina. Sin relative en el padre, el badge se posicionaría respecto al viewport.

B

box-shadow: 0 0 0 1px #10b981, ... Doble sombra con coma. La primera (0 0 0 1px) simula un borde con sombra sin desplazamiento. La segunda es la sombra de profundidad. Se pueden apilar tantas como necesites.

C

position: absolute; top: 1rem; right: 1rem Sale del flujo normal. No empuja a ningún elemento — flota encima del contenido de la tarjeta.

F

flex-grow: 1 en la lista. Llena el espacio disponible. Si una tarjeta tiene más items que otra, la lista crece para ocupar el espacio — el botón queda siempre al fondo, alineado con otras tarjetas del mismo grid.

G-H

li::before { content: "✓" } Pseudo-elemento que inyecta el ícono automáticamente. No necesitas escribirlo en el HTML. La clase .disabled sobreescribe el ::before con "×" — mismo mecanismo, diferente resultado.

¿Cuándo usar cada variante?

🃏 Feature Card — Para mostrar servicios, características o beneficios.

👤 Profile Card — Para equipos, autores, testimonios o directorios.

💰 Pricing Card — Para planes de suscripción o comparativas.

Las tres comparten la misma base: display:flex + flex-direction:column + border-radius.
