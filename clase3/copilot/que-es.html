¿Qué es GitHub Copilot?

El primer asistente de IA masivo para programadores: entiende tu código y sugiere lo que viene a continuación.

AI

Tu Compañero de Programación con IA

GitHub Copilot es una herramienta de IA desarrollada por GitHub y OpenAI que se integra directamente en tu editor de código (VS Code). Analiza el contexto de lo que estás escribiendo y sugiere la siguiente línea, función o bloque de código completo en tiempo real.

Lanzado en 2021 Basado en GPT-4 +1 millón de usuarios Gratuito para estudiantes

¿Qué puede hacer?

Autocompletado Inteligente

Predice tu próxima línea de código basándose en el contexto del archivo, el nombre de funciones y comentarios. Acepta con Tab.

Chat Técnico

Haz preguntas directamente en el editor: explica código existente, pide refactorizaciones o genera componentes completos desde texto.

Código desde Comentarios

Escribe un comentario describiendo lo que quieres y Copilot genera la función completa por ti. El lenguaje natural se convierte en código.

Documentación y Tests

Genera automáticamente comentarios JSDoc, casos de prueba y ejemplos de uso para tus funciones existentes.

Lo que Copilot NO es

No es infalible

Puede generar código con bugs o malas prácticas. Siempre debes revisar y entender el código sugerido antes de usarlo.

No reemplaza aprender

Es un acelerador, no un sustituto. Si no entiendes el código que copia, no podrás mantenerlo ni depurarlo cuando algo falle.

No es un buscador

Copilot genera código basándose en patrones, no busca la documentación oficial. Puede estar desactualizado o ser inexacto.

Cómo Activarlo en VS Code

1

Cuenta GitHub Students

Regístrate en education.github.com con tu correo INACAP para acceso gratuito.

2

Instalar la Extensión

Busca "GitHub Copilot" en el marketplace de extensiones de VS Code e instálala.

3

Iniciar Sesión

Conecta VS Code con tu cuenta de GitHub. El ícono de Copilot aparecerá en la barra inferior.
