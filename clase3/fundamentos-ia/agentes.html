Agentes de IA

Un agente no solo responde — razona, planifica, usa herramientas y actúa para completar tareas complejas.

Chatbot vs Agente

Chatbot / Modelo base

  • Recibe un mensaje, genera una respuesta, termina.
  • No puede acceder a internet, archivos ni APIs.
  • No recuerda conversaciones anteriores por defecto.
  • No puede ejecutar código ni realizar acciones.
  • Ejemplo: ChatGPT sin plugins, Claude sin herramientas.

Agente de IA

  • Recibe un objetivo y trabaja iterativamente para alcanzarlo.
  • Puede usar herramientas: búsqueda web, ejecutar código, escritura de archivos.
  • Mantiene estado y memoria entre pasos.
  • Decide qué hacer en cada paso según el resultado anterior.
  • Ejemplo: Claude Code, Cursor Agent, GitHub Copilot Workspace.
