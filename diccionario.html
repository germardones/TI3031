Diccionario de Etiquetas HTML

Las 100 etiquetas más utilizadas agrupadas por categoría funcional.

1. Estructura y Raíz (4)
<!DOCTYPE html>
Declara que el documento es HTML5.
<html>
El contenedor raíz de todo el documento.
<head>
Contiene metadatos, títulos y enlaces a estilos (no visible).
<body>
Contiene todo el contenido visible de la página.
2. Metadatos y Scripting (8)
<title>
Define el título de la pestaña del navegador.
<meta>
Define metadatos (charset, descripción, viewport).
<link>
Vincula el HTML con recursos externos (como CSS).
<style>
Permite escribir CSS directamente en el HTML.
<script>
Carga o contiene código JavaScript.
<noscript>
Contenido que se muestra si JavaScript está desactivado.
<template>
Contenedor de contenido que no se renderiza al cargar.
<base>
Especifica la URL base para todos los enlaces relativos.
3. Secciones Semánticas (12)
<header>
Cabecera de una página o sección.
<nav>
Bloque de enlaces de navegación.
<main>
El contenido principal y único del documento.
<article>
Contenido independiente y distribuible (post, noticia).
<section>
Sección genérica de contenido relacionado.
<aside>
Contenido tangencial o barra lateral.
<footer>
Pie de página o de una sección.
<h1> a <h6>
Encabezados jerárquicos (del más al menos importante).
<hgroup>
Agrupa varios niveles de encabezados relacionados.
<address>
Información de contacto del autor.
<details>
Crea un desplegable que el usuario puede abrir.
<summary>
El título visible de un elemento <details>.
4. Agrupación de Contenido (11)
<p>
Define un párrafo.
<hr>
Ruptura temática (línea horizontal).
<pre>
Texto preformateado (respeta espacios y saltos).
<blockquote>
Cita en bloque de una fuente externa.
<ol>
Lista ordenada (numerada).
<ul>
Lista no ordenada (viñetas).
<li>
Ítem de una lista.
<dl>
Lista de descripción.
<dt>
Término en una lista de descripción.
<dd>
Descripción del término anterior.
<div>
Contenedor genérico de bloque (usar solo si no hay opción semántica).
5. Semántica de Texto (23)
<a>
Hipervínculo (enlace).
<em>
Énfasis ligero (cursiva).
<strong>
Importancia fuerte (negrita).
<small>
Comentarios secundarios o "letra pequeña" legal.
<s>
Texto que ya no es correcto o relevante (tachado).
<cite>
Referencia a una obra creativa.
<q>
Cita corta en línea.
<dfn>
Indica la definición de un término.
<abbr>
Abreviatura o acrónimo (con atributo title).
<ruby>
Anotaciones para tipografía de Asia Oriental.
<rt>
Pronunciación de los caracteres en <ruby>.
<rp>
Paréntesis para navegadores que no soportan <ruby>.
<data>
Enlaza contenido con un valor legible por máquinas.
<time>
Fecha o hora específica.
<code>
Fragmento de código de computadora.
<var>
Variable en programación o matemáticas.
<samp>
Salida de un programa o sistema.
<kbd>
Entrada de teclado del usuario.
<sub>
Subíndice.
<sup>
Superíndice.
<i>
Texto en cursiva por razones estéticas.
<b>
Texto en negrita por razones estéticas.
<u>
Texto subrayado.
6. Multimedia e Incrustados (11)
<img>
Inserta una imagen.
<iframe>
Incrusta otra página web dentro de la actual.
<embed>
Contenedor para aplicaciones externas (obsoleto en su mayoría).
<object>
Recurso externo (imagen, PDF, plugin).
<param>
Parámetros para etiquetas <object>.
<video>
Reproductor de video.
<audio>
Reproductor de audio.
<source>
Define múltiples recursos multimedia para video/audio.
<track>
Pistas de texto (subtítulos) para video/audio.
<canvas>
Área para dibujar gráficos vía JavaScript.
<map>
Define un mapa de imagen interactivo.
7. Tablas (8)
<table>
Define una tabla.
<caption>
Título o descripción de la tabla.
<thead>
Grupo de encabezados de la tabla.
<tbody>
Cuerpo de la tabla con los datos.
<tfoot>
Pie de la tabla (resúmenes).
<tr>
Fila de la tabla.
<th>
Celda de encabezado.
<td>
Celda de datos.
8. Formularios e Interacción (23)
<form>
Contenedor de un formulario.
<label>
Etiqueta para un elemento de entrada.
<input>
Campo de entrada de datos (múltiples tipos).
<button>
Botón clickable.
<select>
Lista desplegable.
<datalist>
Opciones predefinidas para un <input>.
<optgroup>
Agrupa opciones dentro de un <select>.
<option>
Opción individual en un desplegable.
<textarea>
Campo de texto de múltiples líneas.
<output>
Resultado de un cálculo.
<progress>
Barra de progreso.
<meter>
Medida escalar dentro de un rango conocido.
<fieldset>
Agrupa elementos relacionados en un formulario.
<legend>
Título para un <fieldset>.
<search>
Sección que contiene herramientas de búsqueda.
<dialog>
Caja de diálogo o ventana modal.
9. Otros y Edición (7)
<span>
Contenedor genérico en línea (inline).
<br>
Salto de línea simple.
<wbr>
Oportunidad de salto de línea opcional.
<ins>
Texto insertado (edición).
<del>
Texto eliminado (edición).
<area>
Región dentro de un mapa de imagen (<map>).
<picture>
Contenedor para ofrecer diferentes versiones de una imagen.
